# Traders Switch 2 Alert

트레이더스 5개 지점에서 닌텐도 스위치 2 재고를 확인하고, 매 실행마다 슬랙 웹훅으로 현황을 보내는 개인용 스크립트입니다. 하나라도 재고가 있으면 `@channel` 을 붙여서 보냅니다.

## 감시 대상 지점

- `트레이더스 홀세일 클럽 마곡점 (storeId=2023)`
- `트레이더스 홀세일 클럽 고양점 (storeId=2012)`
- `트레이더스 홀세일 클럽 부천점 (storeId=2017)`
- `트레이더스 홀세일 클럽 김포점 (storeId=2014)`
- `트레이더스 홀세일 클럽 일산점 (storeId=2010)`
